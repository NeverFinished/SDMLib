StudyRightObjectModelNavigationAndQueries

Extend the class model:

How to navigate and query an object model.

Start: Example object structure:

Step 1: Simple set based navigation:

 
      int assignmentPoints = university.getRooms().getAssignments().getPoints().sum();
      
      int donePoints = university.getStudents().getDone().getPoints().sum();

Results in:

      Sum of assignment points: 23. 
      Sum of points of assignments that have been done by at least one students: 15.

Step 2: Rooms with assignments not yet done by Karli:

      
      AssignmentSet availableAssignments = university.getRooms().getAssignments().minus(karli.getDone());
      
      RoomSet rooms = availableAssignments.getRoom();

Results in:

      (senate math 17, gymnasium sports 25)

Step 3: Filter for attribute:

      
      RoomSet rooms17 = rooms.hasCredits(17);

Results in:

      (senate math 17)

Step 4: Filter for attribute greater than:

      
      // Java 8:
      // (Room elem) -> elem.getCredits() > 20
      
      RoomSet roomsGT = rooms.has(rooms.new Condition() {

         @Override
         public boolean check(Room elem)
         {
            return elem.getCredits() > 20;
         }
         
      });
      
      // or easier now
      RoomSet roomsInRange = rooms.hasCredits(21, Integer.MAX_VALUE);

Results in:

      (gymnasium sports 25)
      (gymnasium sports 25)

Step 5: Write operations on sets: 

      
      university.getStudents().withMotivation(42);

Step 6: Rooms with two students that are friends (and need supervision): 

      
      RoomPO roomPO = university.getRooms().startModelPattern();
            
      StudentPO stud1PO = roomPO.hasStudents();      
      
      roomPO.hasStudents().hasMotivation(42).hasFriends(stud1PO);
      
      rooms = roomPO.allMatches();

Results in:

      (gymnasium sports 25)

Step 7: Rooms with two students with low motivation that are friends (and need supervision): 

      
      roomPO = university.getRooms().startModelPattern();
            
      stud1PO = roomPO.hasStudents();      
      
      final StudentPO stud2PO = roomPO.hasStudents().hasMotivation(0, 50);
      
      // Java 8: 
      // stud2PO.has( () -> stud2PO.getMotivation() < 50);
      
      //      stud2PO.has(new GenericConstraint.Condition()
      //      {
      //         @Override
      //         public boolean check()
      //         {
      //            return stud2PO.getMotivation() < 50;
      //         }
      //      });
      
      stud2PO.hasFriends(stud1PO);
      
      rooms = roomPO.allMatches();

Results in:

      (gymnasium sports 25)

Step 8: Rooms with two students without supervision that are friends and add teaching assistance: 

      
      roomPO = university.getRooms().startModelPattern();
            
      stud1PO = roomPO.hasStudents();      
      
      roomPO.hasStudents().hasFriends(stud1PO);
      
      roomPO.hasTas(null);
      
      roomPO.startCreate().hasTas().endCreate();
      
      rooms = roomPO.allMatches();

Results in:

      (gymnasium sports 25)

Step 9: TAs as students in a room: 

      
      roomPO = university.getRooms().startModelPattern();
      
      stud1PO = roomPO.hasStudents();
      
      TeachingAssistantPO taPO = stud1PO.instanceOf(new TeachingAssistantPO());
      
      TeachingAssistantSet taSet = taPO.allMatches();
