Welcome to SDMLib

SDMLib is a light weight modeling library. SDMLib intentionally comes without any tool or editor.

The idea is that you code your model: 



ClassModel model = new ClassModel();

Clazz uni = model.createClazz("University");

Clazz stud = model.createClazz("Student").withAttribute("name", DataType.STRING);

uni.withBidirectional(student, "students", Cardinality.MANY, "almaMater", Cardinality.ONE);

model.generate();

By running model.generate() SDMLib will generate source code from your class model. If you extend your model later on and regenerate, the code generation will identify already existing classes and fields and attributes and leave them untouched. Only new elements will be inserted, carefully.

In addition to class models, SDMLib will also support object diagrams, story pages and model transformations.

  • To learn about class diagrams and code generation, start with StudyRightWithAssignmentsClassGeneration.

  • To learn how to use the generated classes see and to generate test documentation with story pages see StudyRightWithAssignmentsStoryPage

  • For advanced model navigation and model transformations have a look at StudyRightObjectModelNavigationAndQueries

  • To store your object model as Json and to load it again or to send (parts of it) to another process, see JsonPersistency
